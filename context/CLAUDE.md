# Context

Active working context for `caveman`. The CaveKit hierarchy (kits/impl/blueprints)
has been retired and moved to `archive/cavekit/`.

## Active layout

| Dir | Purpose |
|-----|---------|
| `plans/` | Active plans. `cave-v2-best-in-class.md` is the master. |
| `designs/` | Visual design specs (DESIGN.md) referenced by UI work. |
| `refs/` | Reference materials and research briefs. |
| `reports/` | Eval and proof-bench reports. |
| `archive/` | Retired CaveKit kits, plans, impl tracking. |

## Navigation

- Start with the active master plan: `plans/cave-v2-best-in-class.md`.
- UI work: also read `designs/DESIGN.md`.
- Reference materials in `refs/` are stable; don't edit without intent.
- Anything under `archive/` is historical and not load-bearing.
